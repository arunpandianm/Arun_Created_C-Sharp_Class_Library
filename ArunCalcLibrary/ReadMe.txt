How to Add Class Reference:
Step 1: Goto Solution Explorer in Visual Studio
Step 2: Right Click on Reference folder and select "Add Reference"
Step 3: Click on "Browse Tab"
Step 4: Browse to ArunCalc.dll and Click Ok

How to Used it:

Step 1 : type the "using ArunCalc; above the namespace"  //without quotes
Step 2 : Create an object for "ArunCalcLibrary"
		for example : ArunCalcLibrary myObject = new ArunCalcLibrary();
			      myObject.intSum(5,5);

Helper Method of ArunCalcLibrary:

intSum(int, int)		floatSum(float, float)
intDiv(int, int)		floatDiv(float, float)
intMul(int, int)		floatMul(float, float)
intMod(int, int)		floatMod(float, float)
return int;			return float;
